  1. Menginstal dan mengkonfigurasi GNS3 menggunakan GNS3 VM di VirtualBox sebagai server utama.
  2. Membuat topologi jaringan sederhana menggunakan MikroTik CHR (Cloud Hosted Router) sebagai router dan Debian sebagai PC.
  3. Mengimplementasikan DHCP server pada MikroTik dan mengatur Debian sebagai DHCP client.

Persiapan

  1. Komputer dengan spesifikasi minimal:
    • Prosesor multi-core (disarankan 4 core atau lebih).
    • RAM 8GB (minimal 4GB untuk VM).
    • Ruang penyimpanan 20GB.
  2. Software:
    • VirtualBox (terbaru) dan VirtualBox Extension Pack.
    • File GNS3 VM (format .ova).
    • GNS3 GUI (versi terbaru).
    • Image MikroTik CHR dan Debian (unduh dari situs resmi).
    • WinBox untuk remote MikroTik.
  3. Sistem operasi host: Windows, macOS, atau Linux.
